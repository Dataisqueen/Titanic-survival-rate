<h1> Titanic Survival Rates </h1>
A Kaggle challenge. 
The aim is to engage apropiate machine learning algorythims to predict which passangers of the Titanic would survive the catastrophe.

  <li> <b>MatplotLib</b> - visualisations </li>
  </ul>
